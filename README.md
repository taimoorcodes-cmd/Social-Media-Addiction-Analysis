
# Social Media Addiction Analysis


## Project Overview
Analysis of 10,000 users to understand 
social media addiction patterns and 
identify key factors affecting addiction.


## Tools Used
- Python (Pandas) - Data Cleaning
- MySQL - SQL Analysis  
- Power BI - Dashboard


## Dataset
- 10,000 rows
- 23 columns
- Source: Kaggle


## Key Findings
1. Heavy users score 67.00 vs 42.95 
   for low usage users
2. High dopamine users score 68.66 
   vs 46.94 for low dopamine users
3. High addiction users sleep quality 
   5.83 vs 7.30 for low addiction users
4. TikTok users show higher addiction 
   than Instagram users
5. Netherlands leads with 116 high 
   addiction users
